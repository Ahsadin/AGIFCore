# AGIFCore

AGIFCore is the final bounded-intelligence baseline in this repository.

It should be read in three separate layers:

- what the project built
- what the project proved
- what the project did not prove

The build scope is larger than the final proven claim.
That is intentional and truthful.

## A. What AGIFCore Is

AGIFCore is a local-first, governed intelligence architecture/runtime/core.

It was built as a multi-phase system, not as a single chat shell.
The repository includes architectural foundations, cognition layers, runtime layers, sandboxing and scaling layers, and final proof and publication layers.

The final closeout claim is narrower than the full build:

- supported and closed: bounded intelligence baseline
- unproven and deferred: broad open-ended non-neural chat

## B. What AGIFCore Built

AGIFCore built a full multi-phase stack that includes:

- kernel and workspace foundations
- cells, tissues, structure, and bundle composition
- memory planes and lifecycle controls
- graph and knowledge structures
- world-model and simulation layers
- conversation and answer-flow layers
- science and world-awareness layers
- rich expression and composition layers
- critique and meta-cognition layers
- governed self-improvement scaffolding
- structural growth controls
- product runtime and UX surfaces
- sandbox, profile, and scale controls
- proof, closure, and public evidence layers

In other words, AGIFCore built more than the final bounded proof exercised directly.
The repository should not be read as if it only contains a small benchmark-facing runtime.

## C. What AGIFCore Proved

The final proven claim is narrower than the build:

- AGIFCore is supported and closed as a bounded intelligence baseline

That claim is supported by:

- a frozen bounded-intelligence gate: `49/50`, `98%`, `0` hard fails
- a paraphrased shadow benchmark: `50/50`, `100%`, `0` hard fails
- a cleared anti-shortcut audit

The supported bounded behaviors include:

- supported local system, project, runtime, evidence, and release questions
- bounded reasoning and simple math
- follow-up binding from prior turn state
- contradiction and ambiguity handling
- bounded current-world estimates with explicit uncertainty
- honest fail-closed behavior on unsupported requests
- machine-readable proof surfaces for audit and review

## D. What AGIFCore Did Not Prove

AGIFCore did not prove:

- broad open-ended non-neural chat intelligence
- unrestricted conversational intelligence
- unrestricted live world knowledge

The broad-chat path remained unproven and was explicitly not promoted into the final claim.
