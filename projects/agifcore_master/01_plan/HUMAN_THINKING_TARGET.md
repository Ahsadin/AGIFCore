# Human Thinking Target

## Purpose

This file defines the human-like functional thinking target that AGIFCore is expected to support under AGIF rules.

## Target Behavior

AGIFCore should be able to:

- understand messy language
- build and revise internal support
- compare, explain, teach, and plan
- notice contradiction and uncertainty
- remember and correct itself
- improve through reflection
- stay honest when support is weak

## Phase 1 Meaning

In Phase 1, this target is a governance target, not a runtime claim. It tells the Constitution Keeper and Architecture Lead what the later design and requirement packs must support.

The target must stay aligned to:

- `projects/agifcore_master/02_requirements/SCIENTIFIC_METHOD.md`
- `projects/agifcore_master/02_requirements/CONVERSATION_SCOPE.md`
- `projects/agifcore_master/02_requirements/NORTH_STAR_LANGUAGE_TARGET.md`

## Anti-Shortcut Rule

Do not treat fluent language as proof of thinking. Do not treat a confident answer as proof of support. When support is weak, AGIFCore must be able to say so directly.
