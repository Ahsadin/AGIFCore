# AGIFCore Bounded Intelligence Claim Boundary

Status: `candidate_closeout_boundary`

## Goal

This document defines the only truthful closeout claim allowed for AGIFCore in the current repo state.

AGIFCore is not being closed as broad open-ended chat intelligence.
AGIFCore may only be closed, if it closes, as a bounded intelligence core.

## What AGIFCore Proves Only If The Bounded Gate Passes

AGIFCore may claim bounded intelligence only if the bounded-intelligence gate passes honestly and machine-readably.

That bounded claim means AGIFCore proves all of the following through the real runtime path:

1. It answers supported local system, project, runtime, evidence, and manifest questions from real local truth.
2. It performs bounded structured reasoning and simple math/logic without collapsing into generic fallback.
3. It carries follow-up questions through prior-turn state.
4. It handles contradiction, ambiguity, and missing support honestly through clarify, abstain, or search-needed behavior.
5. It can produce some bounded current-world estimates when live truth is unavailable, while clearly labeling uncertainty.
6. It fails closed honestly on unsupported questions without hallucinating.
7. It exposes inspectable phase-chain evidence for those turns.

## What AGIFCore Does Not Prove

AGIFCore does not prove:

- broad open-domain conversational intelligence
- unrestricted world knowledge
- human-like general chat intelligence
- general AGI
- universal question answering
- reliable current-world factual knowledge without live support

## Broad Chat Claim Status

Broad open-ended chat intelligence remains:

- `unproven`
- `deferred`
- `not claimable in closeout surfaces`

If the bounded gate fails, the correct outcome is not to restate a broad-chat goal more strongly.
The correct outcome is to keep Phase 15 and Phase 16 open and report the failure honestly.

## Allowed Closeout Language

Allowed closeout language on pass:

- `AGIFCore is closed as a bounded intelligence baseline`
- `AGIFCore proves bounded local intelligence behavior under the frozen gate`
- `broad chat intelligence remains unproven/deferred`

Forbidden closeout language:

- `AGIFCore proves broad chat intelligence`
- `AGIFCore proves open-domain intelligence`
- `AGIFCore is AGI`
- `AGIFCore has human-like general intelligence`

## Evidence Requirement

This claim boundary is only satisfied if all of the following exist:

- frozen bounded benchmark
- executable bounded gate verifier
- machine-readable per-turn gate report
- machine-readable gate summary
- machine-readable failure summary if the gate fails
- role-separated audit
- Governor verification
- validation surface
- consistent claim wording across Phase 15 and Phase 16 closeout files

Summary text alone is never enough.

## Closeout Rule

- If the bounded gate fails: Phase 15 stays `open`, Phase 16 stays `open`, and AGIFCore is not closed.
- If the bounded gate passes: Phase 15 and Phase 16 may be closed only as bounded intelligence, never as broad chat intelligence.
