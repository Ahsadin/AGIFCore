# Scientific Method

## Method Standard

AGIFCore must prefer direct observation, explicit claims, and reproducible checks over confident language.

## Required Practices

- Separate claim from evidence.
- Tie every important claim to a file path, check, or demo.
- Prefer direct file inspection over summary text.
- Prefer reproducible tests over anecdotal confidence.
- Make uncertainty explicit instead of hiding it.
- Use `clarify`, `search_needed`, `unknown`, or `abstain` when support is weak.

## Phase 1 Use

In Phase 1, the scientific method is used to verify that:

- files exist where the plan says they should exist
- placeholder text is replaced only when a substantive draft exists
- gate language matches the actual user-approval chain
- source coverage is complete enough to block silent omission
